- # 1. Классификация профессий
- ## 1.1. Основные профессии
- ### Механик
- *Направления*: математика, физика, работа механизмов
- *Базовые навыки*:
	- Ремонт инструментов
	- Создание простых механизмов
- *Специализации*:
	- **Инженер**:
		- Создание сложных механизмов
		- Более успешные процес создания предметов и построек
	- **Электротехник**:
		- Крафт генераторов и всего что всязанно с током
		- Ремонт энергооружия
	- **Автомеханик**:
		- более дешевый ремонт транспорта
		- бережное использование транспорта
- ### Солдат
- *Направления*: боевые искусства, тактика ведения боя
- *Базовые навыки*:
	- Ближний бой
	- Использование простого оружия
- *Специализации*:
	- **Убийца** :
		- Бонус к характеристикам скрытого оружия
		- Критические атаки в спину
		- Незаметность в тени
	- **Защитник**:
		- Бонус к характеристикам оружию ближнего боя
		- Контратаки при блоке
	- **Стрелок**:
		- Бонус к характеристикам огнестрельного оружия
		- Стабилизация прицела
	- **Мастер ловушек**:
		- Бонус к характеристикам ловушек
		- Создание на ловушки перед началом боя
- ### Торговец
- *Направления*: психология, торговые отношения
- *Базовые навыки*:
	- Торговля
	- Убеждение
- *Специализации*:
	- **Дипломат**:
		- Скидки у мирных NPC
		- Больше награды за квесты
		- Меньше шанс нападения бандитов на персонажа
	- **Манипулятор**:
		- Убеждение на врагов на поле боя (только разумных)
		- Доступ к запрещённой информации (альтернативные диалоги)
	- **Контрабандист**:
		- Торговля запрещёнными / опасными товарами
		- Доступ к чёрному рынку
		- Негативная репутация у мирных NPC
- ### Ученый
- *Направления*: психология, торговые отношения
- *Базовые навыки*:
	- Анализ
	- Создание рецептов
- *Специализации*:
	- **Биолог**:
		- Лекарства с усиленным эффектом
		- Ускоренное выращивание культур
	- **Зоолог**:
		- Боевые питомцы
		- Ускоренная дрессировка
	- **Химик**:
		- Взрывчатка с увеличенным радиусом
		- Синтез редких ядов
- ## 1.2. Смежные профессии
- Комбинации основных направлений (например, *Инженер-Химик* для создания взрывчатки).

- # 2. Дерево профессий
- Каждая профессия — это **древовидная структура**, где:
- **Узлы** = навыки, рецепты или механики
- **Ветви** = специализации
- **Корень** = базовая профессия
- ## 2.1. Принципы прокачки
- **Открытие узлов**:
	- Выполнение уникальных действий
	- Чем мощнее узел, тем сложнее требования (комбинация действий + ресурсы).
- **Сложность узлов**:
	- **Базовые**: 3-15 однотипных действий (например, *крафт верёвки*).
	- **Продвинутые**: Цепочка из 2+ разных действий (например, *разобрать 3 машины* + *создание 20 единиц металла*).
	- **Элитные**: уникальные квесты и/или редкие ресурсы (например, *квест* + *10 транзисторов*).
- ---
- # 3. Условия для узлов
- ## 3.1. По типу профессии
- | Профессия | Типовые условия прокачки | Пример узла |
  | ---- | ---- | ---- |
  | *Механик* | Ремонт, сборка механизмов | Создание генератора |
  | *Солдат* | Победы в боях, установка ловушек | Бонус к броне |
  | *Ученый* | Исследование предметов/существ | Рецепт антидота |
- ## 3.2. По сложности
- **Уровень 1**: 5-15 однотипных действий
- **Уровень 2**: Цепочка из 3+ специфичных действий
- **Уровень 3**: Квест + редкие ресурсы
- ---
- # 4. Особые механики
- ## 4.1. Пересечение ветвей
- Некоторые узлы требуют навыков из **разных профессий**:
- **Биоинженер** (*Ученый-Биолог* + *Механик*) → протезы с усилением.
- **Тактик** (*Боец* + *Торговец*) → бонусы к планированию атак.
- ## 4.2. Ограничения
- Чем больше активных профессий, тем сложнее открывать в них новые узлы.
- Одновременно без штрафа можно развивать **2 продвинутые** и **1 смежную** профессию. Каждая следующая будет накладывать *штраф на себя*, и *уменьшенный штраф* на уже *активные профессии*.
- *Базовые профессии не подвержены штрафам!*
- ---
- # 5. Стратегии развития
- **Ранняя игра**:
	- Откройте базовые узлы для выживания (аптечки, инструменты).
- **Поздняя игра**:
	- Комбинируйте профессии для уникальных сборок (например, *Химик* + *Инженер* = огнемёт).
- **Эксперименты**:
	- Некоторые узлы открывают альтернативные концовки (например, *Дипломат* → мирный финал).
